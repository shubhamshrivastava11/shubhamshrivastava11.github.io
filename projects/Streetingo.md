Streetingo is a mobile application designed to improve the user journey by offering real-time vendor tracking and enhanced customer experience.

<hr>

<pre>
Overview
This project involved leading the development of the Streetingo mobile app MVP, with a focus on real-time vendor tracking, user experience, and seamless communication between users and vendors. The app was successfully deployed on the Google Play Store, driving significant user engagement.

Features
- Real-time Vendor Tracking: Developed features that allow users to track vendors in real-time, enhancing the overall customer experience.
- Daily Active Users Growth: Increased daily active users by 15% through feature enhancements and improved user journeys.
- Prototyping & Design: Created interactive prototypes using Figma, enabling better communication between developers and designers, and reducing design iteration cycles.
- User Research: Conducted in-depth user research and competitive analysis, translating insights into actionable user stories for future product improvements.
- Product Roadmap Alignment: Collaborated with stakeholders to align the product roadmap with both business objectives and user needs.

Tech Stack
- Mobile App Development: Java (Android)
- Real-time Tracking: Firebase Realtime Database, Google Maps API
- Prototyping: Figma
- User Research: Surveys, Interviews
- Deployment: Google Play Store

Key Achievements
- MVP Development: Led the development of the MVP version of Streetingo, focusing on core functionality and user engagement.
- Daily Active Users: Improved daily active user metrics by 15% post-launch.
- Real-time Vendor Tracking: Implemented real-time tracking features, improving user engagement and satisfaction.
- Prototyping & Design: Designed interactive Figma prototypes to communicate design ideas clearly, reducing feedback cycles.
- Product Roadmap: Worked closely with stakeholders to ensure product development aligned with user needs and business goals.
- Google Play Store Deployment: Successfully deployed Streetingo on Google Play Store, ensuring smooth onboarding and scaling.

</pre>
